API access to OWN-PT

Three URLs in this site also serve back s-expressions instead of HTML if you pass parameter sexp=yes. They are:
  • Document search: https://openwordnet-pt.org/search
  • Synset page: https://openwordnet-pt.org/synset
  • Activity search: https://openwordnet-pt.org/search-activities
All the parameters accepted via browser are also recognized via web service call. Examples: 
 curl "https://www.openwordnet-pt.org/search?search_field=all&term=carro&sexp=yes"
